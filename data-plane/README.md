# Proyecto DDoS-SDN

Deteccion y mitigacion de ataques DDoS en redes SDN mediante Machine Learning
con entropia de Shannon. Maestria en Ciberseguridad.

## Arquitectura
- VM-1 (SDN-DATA, 10.10.10.30): plano de datos -- Mininet + Open vSwitch (OpenFlow 1.3)
- VM-2 (SDN-CONTROL, 10.10.10.40): plano de control -- Ryu + ML

Ver Documento Rector para detalle completo.
